## Context

Follow-up closes the case. The traveler leaves with structured discharge instructions, a safety-net phone check is made for moderate-and-above outpatient cases, and any indicated return visits are scheduled inside the original package. Closure is a joint act between HMC Operations and ADAC, on agreed clinical and administrative criteria. Aggregate data from every closure feeds the monthly performance report.

## Key points

- **12-hour safety-net call.** A scheduled phone check for moderate-and-above outpatient cases catches deterioration before it becomes a second case.
- **Return visits inside the package.** Suture removal, dressing changes and re-assessments are included in the original package price when clinically indicated.
- **Pediatric follow-up by default.** Cases involving children get a next-day phone check as a standing protocol, not a request.
- **Joint closure with ADAC.** The case is closed against agreed clinical and administrative criteria, so both sides see the same end state.
- **Feeds the monthly performance report.** Each closure contributes to aggregate KPIs — case volume, severity mix, mode distribution, escalation rate, closure time.

## What this means for ADAC

Follow-up is where the package-based model proves its operational value — the case ends cleanly, the traveler experience is consistent, and the data that closes one case helps improve the next hundred. ADAC receives a closed case file and a monthly performance report it can use as a board-grade evidence base, not a marketing recap.
