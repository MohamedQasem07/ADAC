## Question

Do we confirm EUR 250 as the pre-authorization threshold default for the pilot, and how do escalation cases sit above it?

## HMC suggested position

- **Under threshold proceeds directly.** Cases below EUR 250 move to treatment under the agreed package framework without a separate approval cycle.
- **Over threshold triggers explicit GOP.** Cases above the threshold route through the agreed channel for a formal Guarantee of Payment before treatment proceeds.
- **Escalation cases bypass the threshold.** All red-flag and time-critical cases get immediate notification regardless of value, with documentation following the clinical event.
- **Threshold reviewed against pilot data.** The cut-off is revisited at the October joint review using actual claim data, not assumptions.

## What we need ADAC to confirm

Confirm the EUR 250 pre-authorization threshold as the pilot default and the named claims contact who receives GOP requests above the line.

## Risk / note

The threshold can move post-pilot if claim data supports a different cut-off; the pilot itself runs on the agreed default for consistency.
