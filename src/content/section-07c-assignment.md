The case coordinator confirms package code, mode of delivery, and target dispatch time with ADAC. Pre-authorization happens on a single number from the published catalogue.

- Package code chosen (e.g. HMC-GI-02) — single decision against the catalogue
- Mode of service confirmed: clinic / hotel-room / mobile unit
- Pre-authorization is a yes/no on the package; no line-item negotiation
- Dispatch target communicated to ADAC and to the AG Holder
- Physician and nurse begin movement; equipment loaded
