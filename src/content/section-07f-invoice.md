## Context

Invoicing is mechanical because the scope is fixed. Each case settles on the single package code assigned at §7.3, and the invoice carries that code and the corresponding price from the published catalogue. Mode of delivery does not change the number. If a case genuinely required escalation beyond the package, the escalation is quoted separately and marked clearly on the document — never folded into the main package price.

## Key points

- **One package, one number.** A single line item per package, mapped to the catalogue code (for example HMC-GI-02, HMC-OR-05).
- **No itemized consumables.** Gauze, sutures, IV fluids and standard medication sit inside the package price by design.
- **Mode-neutral pricing.** Clinic, hotel room and mobile unit settle at the same package price for the same clinical work.
- **Escalation quoted separately.** When clinical reality exceeds the package, the additional scope is a clearly marked, separately priced line.
- **Same-day issuance.** The invoice is generated and dispatched via the agreed channel on the day of treatment, in EUR.

## What this means for ADAC

A predictable, reconcilable cost line. ADAC can match the invoice to the pre-authorized package code in one step, which removes the largest source of friction in conventional fee-for-service claims — itemized review. The claims cycle shortens, disputes drop, and the cases ADAC has to escalate are the ones genuinely worth escalating.
