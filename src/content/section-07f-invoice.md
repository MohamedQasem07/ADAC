The invoice is mechanical because the scope is defined. One package, one number, one line item — mapped to the catalogue code.

- Invoice issued same day via the agreed channel
- One line item per package; no itemized consumables
- Each line references the catalogue code (HMC-GI-02, HMC-OR-05, …)
- Direct billing to ADAC via Guarantee of Payment
- Currency: EUR; payment terms agreed in the partnership contract
