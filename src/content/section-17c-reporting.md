## Question

Same-day case summary, weekly digest, and monthly performance report — do we confirm this three-cadence model for the pilot?

## HMC suggested position

- **Same-day case summary on closure.** Issued for every case the moment it closes, with the standard fields ADAC claims needs for processing.
- **Weekly operational digest.** Aggregates case volume, severity mix, and exceptions into one operational view delivered each week.
- **Monthly performance report.** Covers response time, documentation completeness, and escalation rate against the agreed targets.
- **Quarterly business review on request.** Available when ADAC wants a deeper read on trends, mix shifts, or pilot trajectory.

## What we need ADAC to confirm

Confirm the three-cadence reporting model and flag any format-specific fields ADAC claims needs included, with the layout locked by 24 June.

## Risk / note

If ADAC requires a custom format, the layout has to be locked by 24 June for the July go-live; later changes push the first stable report into August.
