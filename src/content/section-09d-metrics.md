The same metrics are reported every month so the trend line is easy to read. No moving definitions.

- SLA hit-rate: % of cases meeting acknowledgement, triage, and dispatch targets
- Package mix: count of cases per category and per package code
- Escalation rate: % of cases escalated to ER pathway
- Average length of treatment for outpatient cases
- Same-day report dispatch rate
- AG Holder feedback summary (where collected, anonymized)
